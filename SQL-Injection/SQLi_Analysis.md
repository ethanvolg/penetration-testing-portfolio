# SQLi 심층 분석: 버그 바운티 사례로 보는 개발자 실수 패턴

## 1. 분석 개요
실제 취약점이 나타나는 패턴을 알아보기 위해 Hackerone 플랫폼에서 공개 보고서를 바탕으로 제작되었습니다. 자료는 추천수를 가장 많이 받은 상위 보고서 30개 이상을 정리했습니다.
버그 헌터가 어떤 매커니즘으로 해당 취약점을 찾은 건지 정확한 경위는 알지 못하지만, 플랫폼의 성격, 기능, 취약점 위치 등의 정보를 조합해서 어떤 식의 접근이 가능했을지 추론하게되었습니다.
